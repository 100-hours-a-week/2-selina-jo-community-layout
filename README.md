### 폴더 구조

```
📂 community-layout
 ├── 📂 api            # 백엔드와 통신하는 코드 (API 호출)
 │    ├── signup.js
 ├── 📂 assets         # 이미지, 아이콘 파일 저장
 │    ├── profile-placeholder.png
 ├── 📂 components     # 재사용 가능한 컴포넌트
 │    ├── inputField.js
 ├── 📂 config         # 설정 관련 파일
 │    ├── settings.js
 ├── 📂 hooks          # 커스텀 훅 (사용자 입력 관리)
 │    ├── useFormHandler.js
 ├── 📂 layout         # 전체 구조 정의
 │    ├── header.js
 ├── 📂 pages          # 실제 HTML 페이지
 │    ├── login.html
 │    ├── signup.html
 ├── 📂 styles         # CSS 파일 저장
 │    ├── login.css
 │    ├── signup.css
 ├── 📂 scripts        # JS 파일 저장
 │    ├── login.js
 │    ├── signup.js
 ├── 📂 utils          # 공통 유틸리티 함수
 │    ├── validation.js
 ├── 📂 store          # 상태 관리 관련 파일
 │    ├── store.js
 ├── 📂 routes         # 라우팅 관련 코드
 │    ├── navigation.js
 ├── 📜 index.html      # 메인 파일 (선택사항)
 ├── 📜 main.js         # 전체 초기화 스크립트
```
